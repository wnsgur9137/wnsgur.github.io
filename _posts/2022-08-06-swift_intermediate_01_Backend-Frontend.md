# Frontend and Backend

## Client-Server Model

Client = 서비스를 사용하는 사용자 혹는 사용자가 사용하는 기기, 인터넷 등을 일컫는 말이다.  
Server = 서버, 데이터베이스 등을 말한다.

<br>
<br>

## Frontend and Backend

Frontend = 프로세스의 처음 단계  
Backend = 프로세스의 마지막 단계

즉, Frontend는 사용자의 행위(입력, 삭제, 저장 등)를 처리하는 것을 의미한다.
